In Networks, an IP address is used to identify and connect devices to the Internet, and the new standard is IPv6.
A domain name is the human-readable version of an IP address. Some basic commands are whois (which identifies the
domain owner), ping (which checks the connectivity or time it takes data to move across a network), traceroute
(which identifies the nodes along the path to the destination), and curl (which can function for a variety of requests
from resources on the network such as fetching a resource and displaying it in the shell). In the advanced network tools
section, SSH is a tool to connect to other computers on the network. SSH can allow you to edit files, database or pull
changes to a git repository on a web server. SFTP allows you to send files across a network. Tcpdump can allow you to
monitor network activity or capture data packets.
